Deep Learning for Brain Imaging

Problem: Existing neurology studies require advanced tools to visualize and interpret brain scan data.

Solution: I developed a CNN in Python using TensorFlow, enabling real-time analysis of fMRI images and generating 3D reconstructions within a VR environment for deeper insights.

Technologies: Python, TensorFlow, Keras, Unity/Unreal Engine, VR Headsets (HTC Vive), GitHub for collaboration.

Results: Improved data analysis speed by 30%, giving researchers immediate visual feedback on neurological changes. This approach could significantly reduce time-to-discovery in neural research.

Challenges: Dealing with high-dimensional brain data required specialized preprocessing. I employed data augmentation and domain adaptation to handle limited labeled images. Real-time VR rendering also demanded optimization techniques.

Impact: The project paves the way for more interactive and educational neuroscience applications, bridging the gap between research labs and immersive user experiences.

Movie Genre Classifier

Problem: Existing movie classification often relies on metadata like titles and plot summaries, ignoring valuable visual cues from posters.

Solution: Built a CNN in Python (TensorFlow & Keras) to classify movie posters into multiple genres. Scraped thousands of images from IMDb for training.

Technologies: Python, TensorFlow, Keras, Beautiful Soup, OpenCV, NumPy, Matplotlib.

Results: Achieved an F1-score of 0.75—significantly higher than baseline approaches relying on text alone.

Challenges: Class imbalance. Overcame this with data augmentation (rotation, flipping, cropping) and careful architecture tuning (dropout, regularization).

Impact: Demonstrates the power of visual features in classification tasks; could be adapted for recommendation systems in streaming services.

VR Brain Installation

Problem: Creating an engaging way for audiences to learn about complex brain functions using actual neuroscience data.

Solution: Partnered with a neuroscience PhD candidate to develop a VR experience using brain scan data. Converted neural imaging files into VR-friendly formats, ensuring both scientific accuracy and immersive design.

Technologies: Unreal Engine, C++, Python, Maya, AI-driven plugins.

Results: Provided an educational and therapeutic tool that helps viewers visualize real neural pathways, winning first prize at a local Sci-Art expo.

Challenges: Balancing aesthetic VR design with accurate representation of scientific data; carefully tested user interfaces for intuitive interactions.

Impact: Potential to expand into hospitals or museums, bridging academic research and public engagement.
